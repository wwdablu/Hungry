# Hungry

Upgrading the application to Kotlin and matching the UI as of Zomato release in Dec, 2020

**__Libraries used during development:__**  

* Recycler View  
* Card View  
* Design Library

    * CoordinatorLayout
* Retrofit  
* RxJava
* Glide
* Hilt
* Room
* Timber
